
## 🚀 Technologies Used

* Backend: Python (Flask / FastAPI / Django – update based on your project)

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/jayesh-shendurnikar-2001/ai_monk_assignment.git
cd ai_monk_assignment
```

---

### 2. Setup Frontend

```
cd frontend
npm install
npm start
```

---

### 3. Setup Backend

```
cd backend
pip install -r requirements.txt
python app.py
```
